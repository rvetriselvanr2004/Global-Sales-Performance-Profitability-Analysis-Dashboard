# Global Sales Performance & Profitability Analysis Dashboard

## Project Overview
The Global Sales Performance & Profitability Analysis Dashboard is a data analytics project designed to analyze worldwide sales data and generate valuable business insights. The main objective of this project is to evaluate sales performance across different regions, product categories, and time periods to identify trends, profitable markets, and growth opportunities.

This project demonstrates the complete data analysis workflow including data cleaning, data transformation, and data visualization using modern analytics tools.

## Objectives
- Analyze global sales performance
- Identify top-performing products and regions
- Understand revenue and profit trends
- Provide insights for data-driven business decisions

## Dataset Description
The dataset contains sales transaction data from different countries and includes the following fields:

- Order ID
- Order Date
- Ship Date
- Country / Region
- Product Category
- Units Sold
- Unit Price
- Total Revenue
- Total Cost
- Total Profit

## Data Cleaning Process
Data preprocessing was performed using Python and Pandas to ensure data quality.

Steps performed:
- Converted date columns to proper datetime format
- Removed duplicate records
- Checked and handled missing values
- Verified revenue and profit calculations
- Prepared cleaned dataset for analysis

## Tools & Technologies
- Python (Pandas) – Data Cleaning
- Microsoft Excel – Data exploration
- Power BI – Data visualization and dashboard creation
- GitHub – Project version control and sharing

## Dashboard Features
The Power BI dashboard includes the following insights:

- Total Revenue KPI
- Total Profit KPI
- Total Orders KPI
- Profit Margin KPI
- Sales Trend Over Time
- Top Products by Revenue
- Sales by Product Category
- Regional Sales Distribution

Interactive filters allow users to explore data by:
- Year
- Region
- Product Category

## Key Insights
- Identifies high-performing regions and markets
- Highlights the most profitable products
- Shows seasonal sales patterns
- Helps businesses optimize their sales strategies
